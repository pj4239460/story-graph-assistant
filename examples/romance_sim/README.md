# 樱花树下的约定

> 都市恋爱模拟示例项目 - 校园青春恋爱，三位女主角的浪漫故事

## 📖 故事概述

在樱花盛开的春天，你转学来到樱华学园。命运让你遇到了三位性格迥异的女生——高冷的学生会长夏目凛、活泼开朗的运动少女樱井美羽，以及文静神秘的文学少女月宫雪。

你的每一个选择都将决定与谁走得更近，最终谁会成为你的恋人？

## 🎭 角色介绍

### 可攻略角色

#### 夏目凛 - 完美会长线
**属性**：高冷、完美主义、隐藏温柔、钢琴高手

**性格**：外表冷艳，内心温柔。作为学生会长，她总是保持完美形象，但也因此感到压力和孤独。钢琴是她唯一能卸下防备的时刻。

**攻略要点**：
- 看穿她的"完美面具"，关注她的真实感受
- 给她空间展现脆弱的一面
- 欣赏她的音乐才华
- 理解她的责任感和压力

**关键场景**：
- 食堂偷吃炸鸡的可爱一面
- 音乐教室的私人演奏会
- 月光下的告白

#### 樱井美羽 - 青梅竹马线
**属性**：开朗、直率、运动健将、吃货

**性格**：活力四射，笑容灿烂。篮球队队长，喜欢美食，对朋友非常忠诚。看似大大咧咧，但在喜欢的人面前会害羞。

**攻略要点**：
- 欣赏她的活力和直率
- 支持她的运动梦想
- 一起享受美食
- 参加她的比赛为她加油

**关键场景**：
- 校外拉面约会
- 篮球场上的精彩表现
- 夕阳下的心动瞬间
- 篮球场上的告白

#### 月宫雪 - 文学少女线
**属性**：文静、敏感、才华横溢、神秘

**性格**：安静内向，喜欢读书写作。常独自待在图书馆，给人神秘感。渴望被理解，一旦打开心扉就会变得话多。

**攻略要点**：
- 和她分享文学话题
- 理解她的孤独和敏感
- 成为她的灵感来源
- 耐心倾听她的想法

**关键场景**：
- 图书馆的初遇
- 阳光下写作的美丽画面
- 分享她的小说手稿
- 以你为原型的爱情故事

### 辅助角色

#### 林老师
温柔的班主任，为学生们提供建议和引导。在关键时刻会给予玩家提示。

## 📊 故事结构

### 三段式结构

```
第一章：初遇（场景 1-4）
  - 樱花树下的相遇
  - 第一印象的建立
  - 初步好感度获取

第二章：深入了解（场景 5-12）
  - 了解每个角色的真实一面
  - 好感度积累
  - 路线分化

第三章：心意相通（场景 13-15）
  - 告白场景
  - 做出最终选择

终章（场景 16-19）
  - 4种不同结局
  - 每条线的完整故事
```

## 🎯 设计亮点

### 1. 经典Galgame要素
- **多女主选择**：三位风格完全不同的女主角
- **好感度系统**：通过relationship数值跟踪
- **分支剧情**：早期选择决定后续发展
- **CG时刻**：每条线都有标志性场景

### 2. 角色塑造
每个角色都有：
- **表面性格 vs 真实性格**：凛的完美面具、美羽的细腻、雪的话多
- **个人梦想**：音乐、体育、文学
- **成长弧光**：通过恋爱克服内心障碍
- **独特魅力点**：琴声、笑容、文字

### 3. 选择设计哲学
```
第一印象（场景1）→ 决定初始好感
早期互动（场景2-9）→ 积累好感度
分岔点（场景10）→ 确定主线
深入交流（场景11-15）→ 触发告白
最终选择（场景13-15）→ 决定结局
```

### 4. 情感节奏控制
- **初遇**：轻松愉快，建立第一印象
- **了解**：逐渐深入，发现角色魅力
- **心动**：关键时刻，情感升温
- **告白**：高潮场景，情绪爆发
- **结局**：甜蜜收尾，展望未来

## 🌟 4种结局

### 完美结局（3种）

1. **凛的真心**
   - 条件：选择凛线并接受告白
   - 内容：帮助凛卸下完美面具，支持她的音乐梦想
   - 关键词：理解、接纳、共同成长

2. **雪的归属**
   - 条件：选择雪线并接受告白
   - 内容：成为雪的灵感，陪她走出孤独
   - 关键词：陪伴、理解、文学之爱

3. **美羽的笑容**
   - 条件：选择美羽线并接受告白
   - 内容：成为美羽的后盾，见证她的成长
   - 关键词：活力、支持、青春无悔

### 遗憾结局（1种）

4. **错过的春天**
   - 条件：拒绝所有告白或犹豫不决
   - 内容：错过所有机会，独自毕业
   - 关键词：后悔、遗憾、成长

## 💡 教学价值

### 恋爱游戏设计
- **多线程叙事**：三条主线互不干扰但设计对称
- **好感度系统**：通过relationship值跟踪
- **选择后果**：每个选择都有明确的数值影响
- **结局差异化**：不同选择导向不同结局

### 角色塑造技巧
- **对比设计**：三个角色性格、爱好、表达方式完全不同
- **反差萌**：外表vs内心的反差（凛的完美vs脆弱）
- **成长性**：角色在恋爱中成长变化
- **记忆点**：每个角色都有标志性场景

### 情感代入
- **玩家视角**：第一人称，强化代入感
- **选择自由**：玩家可以选择喜欢的类型
- **情感共鸣**：青春、初恋、成长等普遍主题
