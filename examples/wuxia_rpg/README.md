# 江湖风云录 - 武侠RPG示例项目

**类型：** 武侠/角色扮演  
**语言：** 中文  
**难度：** 中级

## 📖 故事简介

你是林傲天，一个被少林收养的孤儿。在追求武学至高境界的过程中，你将面临道德抉择、身世之谜，以及正邪之间的纠葛。

你的每一个选择都会影响故事走向，最终通往不同的结局。

## 🎯 项目特点

### 多重结局系统
- **完美结局：** 武林盟主（化解正邪恩怨）
- **好结局：** 少林方丈 / 少林新星
- **坏结局：** 心魔缠身（走火入魔 / 堕落魔道）

### 复杂角色关系
- 师徒关系：慧空大师（隐藏秘密）
- 情感纠葛：苏婉儿（青梅竹马）vs 红莲（宿命之敌？）
- 身世之谜：魔教的隐秘联系

### 道德抉择
- **诚信 vs 欺骗**：对师父坦白还是撒谎？
- **智慧 vs 蛮力**：如何应对强敌？
- **善意 vs 偏见**：如何对待魔教之人？
- **忠诚 vs 真相**：发现身世后的选择

## 🗺️ 故事流程图

```
少林藏经阁
├─ 坦白 → 师父考验 → 武林大会
│   ├─ 全力一战
│   │   ├─ 改变策略 → 武道领悟 → [继续修炼 / 寻找红莲]
│   │   └─ 孤注一掷 → 两败俱伤 → 复仇之心 → ❌ 坏结局
│   └─ 智取 → 胜利
│       ├─ 查看红莲 → 意外善意 → 继续修炼 → ✅ 好结局（少林新星）
│       └─ 请教心法 → 传承 → ✅ 好结局（少林新星）
│
└─ 撒谎 → 师父怀疑
    ├─ 追上坦白 → （回到坦白路线）
    └─ 偷学 → 走火入魔 → ❌ 坏结局

特殊路线（从武道领悟触发）：
寻找红莲 → 魔教秘密 → 身世抉择
    ├─ 追查真相 → 真相大白 → ⭐ 完美结局（武林盟主）
    └─ 回到少林 → 忠于少林 → ✅ 好结局（少林方丈）
```

## 💡 设计亮点

### 1. 早期选择的长期影响
- scene-001 的诚信选择影响整个故事走向
- 撒谎→偷学会直接导致坏结局（走火入魔）
- 坦白→考验会开启多条发展路线

### 2. 战斗策略的重要性
- 蛮力（孤注一掷）→ 两败俱伤 → 复仇循环
- 智慧（智取 / 以守代攻）→ 胜利 → 更多可能性

### 3. 善意的回报
- 对敌人展现善意（scene-010）开启好结局
- 领悟武道（scene-008）开启隐藏剧情
- 追查真相（scene-012→014）开启完美结局

### 4. 身份认同主题
- 少林弟子 vs 魔教血脉
- 正派教育 vs 真实身世
- 忠诚 vs 真相的两难抉择

## 🎮 如何使用

### 1. 加载项目
```bash
# 在Story Graph Assistant中
1. 点击侧边栏 "打开项目"
2. 选择 examples/wuxia_rpg/project.json
```

### 2. 探索故事
- 从 scene-001（少林藏经阁）开始
- 每个选择都有意义，尝试不同路线
- 使用"Play Path"模式查看不同结局

### 3. 学习要点
- **角色设计**：查看5个角色的特质、目标、恐惧
- **选择设计**：注意道德两难和策略选择
- **结局设计**：4种不同类型的结局
- **故事节奏**：从日常→考验→高潮→真相→结局

## 🎓 教学价值

### 适合学习：
1. **多结局叙事设计**
   - 如何让早期选择产生深远影响
   - 如何平衡好/坏结局的吸引力

2. **角色塑造**
   - traits / goals / fears 的实战应用
   - 角色关系如何影响剧情

3. **道德抉择系统**
   - 无明显对错的选择设计
   - 选择后果的递延效应

4. **武侠题材叙事**
   - 正邪之争的现代诠释
   - 武学与人性的结合
